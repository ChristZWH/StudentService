# Go学生管理系统 - 使用说明

## 项目概要
基于Go语言Gin框架开发的高性能学生管理系统，提供完整的学生信息管理功能，通过以下五个核心API实现全功能覆盖：

### 核心API功能
1. **学生创建API**：
   - 端点：`POST /students`
   - 功能：创建新学生记录
   - 特性：自动记录创建时间，验证输入数据完整性
   - 缓存处理：自动清除学生列表缓存

2. **学生列表查询API**：
   - 端点：`GET /students`
   - 功能：获取所有学生信息列表
   - 特性：使用Redis缓存加速查询，2分钟有效期
   - 缓存策略：缓存未命中时自动查询数据库并更新缓存

3. **学生详情查询API**：
   - 端点：`GET /students/id`
   - 功能：获取指定ID的学生详细信息
   - 特性：使用Redis缓存单个学生数据，5分钟有效期
   - 错误处理：学生不存在时返回404状态码

4. **学生信息更新API**：
   - 端点：`PUT /students/id`
   - 功能：更新指定ID的学生信息
   - 特性：自动更新修改时间字段
   - 缓存处理：更新后自动清除该学生缓存

5. **学生删除API**：
   - 端点：`DELETE /students/id`
   - 功能：删除指定ID的学生记录
   - 特性：事务性操作保证数据一致性
   - 缓存处理：同时清除该学生缓存和列表缓存
